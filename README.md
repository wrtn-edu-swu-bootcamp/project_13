# 송파구 도서관 책크

## 📚 프로젝트 소개

송파구 내 24개 도서관의 도서 소장 및 대출 가능 여부를 한 번에 검색하는 통합 검색 서비스입니다.

특정 도서관에 원하는 책이 없을 때 다른 도서관 홈페이지를 일일이 방문해야 하는 번거로움을 해소하고, 사용자가 가장 빠르게 원하는 책을 찾을 수 있도록 돕습니다.

## 🎯 주요 기능

- 통합 도서 검색
- 24개 도서관 실시간 소장/대출 현황 확인
- 지도 기반 도서관 위치 정보
- 도서관별 상세 정보 제공
