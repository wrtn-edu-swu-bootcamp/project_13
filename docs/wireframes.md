```
[검색 화면] 
    ↓ 검색 버튼 클릭

    ↓ 데이터 수집 완료
```

### 1.2 구성 요소 상세

#### 헤더 영역
- **서비스 로고**: 📚 책크 (좌측 상단)
- **위치 정보**: 📍 송파구 (우측 상단)
  - 위치 권한 허용 시: 현재 위치 표시
  - 위치 권한 거부 시: 기본값 "송파구" 표시

#### 검색 영역
- **제목 입력창**
  - 플레이스홀더: "📖 책 제목을 입력하세요 (선택)"
  - 텍스트 입력 필드
  - 단일 라인 입력

- **저자 입력창**
  - 플레이스홀더: "✍️ 저자명을 입력하세요 (선택)"
  - 텍스트 입력 필드
  - 단일 라인 입력

- **출판사 입력창**
  - 플레이스홀더: "🏢 출판사명을 입력하세요 (선택)"
  - 텍스트 입력 필드
  - 단일 라인 입력

- **안내 텍스트**
  - "ⓘ 최소 1개 이상의 검색어를 입력해주세요"
  - 회색 텍스트로 표시

- **검색 버튼**
  - 텍스트: "🔍 검색하기"
  - 중앙 배치
  - 기본 상태: 활성화 (파란색)
  - 모든 입력창이 비어있을 때: 비활성화 (회색)

#### 최근 검색어 영역
- **제목**: "최근 검색어"
- **검색어 목록** (최대 3-5개)
  - 각 항목 클릭 시 해당 검색어로 재검색
  - 표시 형식: "• 제목" 또는 "• 저자 + 출판사"
  - 로컬 스토리지에 저장

#### 안내 메시지 영역
- 서비스 범위 안내
- 사용 팁 제공

### 1.3 인터랙션

- **입력 필드 포커스**: 테두리 하이라이트 (파란색)
- **검색 버튼 호버**: 버튼 색상 진하게 변경
- **최근 검색어 클릭**: 해당 검색어로 자동 입력 및 검색 실행
- **엔터키**: 검색 실행

### 2.2 구성 요소 상세

#### 헤더 영역
- **뒤로가기 버튼**: ← 뒤로가기 (검색 화면으로 복귀)
- **서비스 로고**: 📚 책크
- **위치 정보**: 📍 송파구

#### 도서 정보 카드
- **좌측**: 책 표지 이미지 (썸네일)
- **우측**: 
  - **제목** (큰 글씨, 볼드)
  - **저자** (중간 글씨)
  - **출판사 · 출판년도** (작은 글씨)
  - **ISBN** (작은 글씨, 회색)
- **하단**: 책 소개 (2-3줄, 요약)

#### 필터 및 정렬 영역
```
┌─────────────────────────────────────────────┐
│  정렬: ○ 거리순  ● 대출가능 우선           │
│  ☑ 대출 가능만 보기                         │
└─────────────────────────────────────────────┘
```

- **정렬 옵션** (라디오 버튼)
  - 거리순: 사용자 위치에서 가까운 순
  - 대출가능 우선: 대출 가능한 도서관 우선 표시
  
- **필터 옵션** (체크박스)
  - 대출 가능만 보기: 체크 시 대출 가능한 도서관만 표시

#### 도서관 카드 (리스트)

각 도서관마다 개별 카드 형태로 표시:

**카드 헤더**
```
┌─────────────────────────────────────────────┐
│  도서관명                    [유형 배지]     │
│  📍 거리 정보                               │
```

**카드 본문**
```
│  [소장 상태 배지] [대출 상태 배지]          │
│  (대출 중인 경우 반납예정일 표시)            │
│                                              │
│  🕒 운영 시간                               │
│  📞 연락처                                  │
│  📌 주소                                    │
```

**카드 하단 (버튼 영역)**
```
│  [상세보기]  [예약하기]                     │
└─────────────────────────────────────────────┘
```

#### 배지 디자인

**도서관 유형 배지**
- `[공공도서관]` - 파란색 배경
- `[스마트도서관 24H]` - 보라색 배경, 24시간 강조
- `[교육청도서관]` - 초록색 배경

**소장 상태 배지**
- `[소장함]` - 녹색 배지 ✓
- `[미소장]` - 회색 배지 ✗

**대출 상태 배지**
- `[대출가능]` - 밝은 녹색 배지, 눈에 띄게 강조 ✓
- `[대출 중]` - 빨간색 배지 ⌛
  - 하단에 "반납예정: YYYY-MM-DD" 표시
- `[관내열람만]` - 노란색 배지 📖

#### 하단 버튼
```
┌──────────────────────────┐
│  📚 다른 책 검색하기     │
└──────────────────────────┘
```
- 중앙 배치
- 클릭 시 검색 화면으로 복귀

### 2.3 인터랙션

- **정렬/필터 변경**: 즉시 도서관 목록 재정렬
- **도서관 카드 호버**: 카드 배경색 살짝 변경 (하이라이트)
- **상세보기 버튼**: 해당 도서관 홈페이지의 도서 상세 페이지로 이동 (새 탭)
- **예약하기 버튼**: 해당 도서관 예약 페이지로 이동 (새 탭)
- **뒤로가기**: 검색 화면으로 복귀 (입력값 유지)

### 2.4 특수 케이스

#### 위치 권한 거부 시
- 거리 정보 표시 안 함
- 정렬 옵션에서 "거리순" 비활성화
- 기본 정렬: 도서관 이름 순 또는 대출가능 우선

#### 모든 도서관이 미소장인 경우
- 안내 메시지 표시
- "송파구 내 모든 도서관에 소장되어 있지 않습니다"
- "다른 지역 또는 상호대차 서비스를 이용해보세요"

## 3. 로딩 화면

### 3.1 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [헤더]                                              │
│  📚 책크                            📍 송파구        │
├─────────────────────────────────────────────────────┤
│                                                      │
│                                                      │
│                                                      │
│                                                      │
│                      ⏳                              │
│                   로딩 중...                         │
│                                                      │
│               ▓▓▓▓▓▓▓▓░░░░░░                        │
│                                                      │
│        도서관 정보를 불러오는 중입니다...            │
│                                                      │
│             예상 소요 시간: 3-5초                    │
│                                                      │
│                                                      │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│                                                      │
│  ⓘ 송파구 내 24개 도서관의                          │
│     실시간 소장 및 대출 정보를 수집하고 있습니다     │
│                                                      │
│                                                      │
│                                                      │
└─────────────────────────────────────────────────────┘
```

### 3.2 구성 요소

- **로딩 스피너**: ⏳ 회전 애니메이션
- **로딩 텍스트**: "로딩 중..."
- **프로그레스바**: 진행률 표시 (애니메이션)
  - 실제 진행률 또는 무한 로딩 바
- **안내 메시지**: "도서관 정보를 불러오는 중입니다..."
- **예상 소요 시간**: "예상 소요 시간: 3-5초"
- **추가 정보**: 수집 중인 데이터 설명

### 3.3 동작

- 검색 버튼 클릭 후 즉시 표시
- 백그라운드에서 도서관 크롤링 진행
- 데이터 수집 완료 시 자동으로 검색 결과 화면으로 전환
- 5초 이상 지연 시 추가 안내 메시지 표시
  - "잠시만 기다려주세요..."
  - "도서관 서버 응답이 지연되고 있습니다"

## 5. 반응형 디자인 고려사항

### 모바일 (스마트폰)
- **화면 너비**: 360px - 414px
- **특징**:
  - 세로 스크롤 중심
  - 터치 인터랙션 최적화
  - 버튼 크기 충분히 확보 (최소 44x44px)
  - 입력 필드 큰 폰트 사용
  - 도서관 카드 전체 너비 사용

### 태블릿
- **화면 너비**: 768px - 1024px
- **특징**:
  - 검색 영역 좌우 여백 추가
  - 도서관 카드 2열 배치 가능
  - 필터/정렬 옵션 상단 고정

### 데스크톱
- **화면 너비**: 1024px 이상
- **특징**:
  - 최대 너비 제한 (1200px)
  - 중앙 정렬
  - 도서관 카드 2-3열 그리드 배치
  - 사이드바 활용 가능

## 7. 접근성 고려사항

### 키보드 내비게이션

- WCAG 2.1 AA 기준 준수
- 색상에만 의존하지 않는 정보 전달
- 아이콘과 텍스트 병행 사용
